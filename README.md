🎤 Voice User Interface (VUI)

## Project Overview
This project is a simple Voice User Interface (VUI) application developed using Python and Tkinter. It demonstrates a basic voice assistant interface where users can interact through a simple graphical interface.

## Features
- Voice User Interface Demo
- Start Listening Button
- Simple GUI using Tkinter
- Beginner-friendly Python project

## Technologies Used
- Python
- Tkinter
